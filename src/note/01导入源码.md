# 3 IDEA 设置

接下来还有两个地方需要设置，才能开始调试源代码。

## 3.1 允许断点进入源码

在 File -> Settings -> Build,Execution,Deployment -> Debugger -> Stepping 中取消`Do not step into the classes`的勾选。

![允许断点进入源码](https://fat-shallot.oss-cn-hangzhou.aliyuncs.com/img/20201112215558.png)

## 3.2 将 jdk 设置为自己粘贴的源码

在 File -> Project Structure -> Platform Settings -> SDKs 里

![设置jdk](https://fat-shallot.oss-cn-hangzhou.aliyuncs.com/img/20201112215600.png)

先删除原有的 src.zip 的路径，然后添加刚刚粘贴进去的源码的路径。
